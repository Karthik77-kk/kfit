# KFitness

Personal fitness tracker app for Android.

This repository is the **public distribution point**: it hosts the marketing
website and the [GitHub Releases](../../releases) that the app's in-app
updater and the download page both pull from. Application source code lives
in a private repository and is no longer published here.

## Download

Grab the latest APK from [Releases](../../releases), or via the
[website](https://karthikfitness.pages.dev).
